# bot1
Our 1-index bot

Setup
- Python Environment:
  - install Poetry (see https://python-poetry.org/)
  - cd into this repository's directory
  - run `poetry shell`
  - run `poetry install`
- Discord Token
  - run `cp .env.example .env`
  - edit `.env` to contain your Discord bot token
- Run
  - `python bot/main.py`
